# Java Weather App

## Description
This is a Java-based weather application that retrieves and displays weather data for a selected location.

## Features
- Fetches real-time weather data
- Displays weather conditions
- Simple user interface

## Technologies Used
- Java
- JavaFX
- Weather API

## What I Learned
This project helped me understand working with APIs, handling data, and building user interfaces in Java.

## How to Run
1. Open the project in Eclipse
2. Run the `Main.java` file
3. Enter a location to view weather data
